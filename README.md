# BHHax

WIP trainer DLL for Brawlhalla.

Compatible with game version `1.5.3`. Dll is x86 dx9

# Building
You may fork this if you're on linux and need to get started with imgui dll it's very easy to build.

See the [workflows](./.github/workflows/)
